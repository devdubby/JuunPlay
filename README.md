# 영화, 드라마 리뷰 웹 페이지

## Movie, TV show 관련 정보, 예고편, 리뷰 등을 볼 수 있는 사이트

## 사용 기술

- React, Redux, Redux-thunk, styled-components
- NodeJS, ExpressJS, Mongoose
- Movie, TV show data는 'https://www.themoviedb.org/' 의 API 사용
